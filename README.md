# 💄 Velora Beauty

Velora Beauty is a full-stack cosmetic e-commerce website developed using PHP, MySQL, HTML, CSS, and JavaScript.

## 🚀 Features

- User Registration & Authentication
- Product Management System
- Product Search Functionality
- Product Detail Page
- Dark Mode Support
- Admin Dashboard
- Add, Edit, and Delete Products
- Image Upload System
- Responsive Design

## 🛠 Technologies Used

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- XAMPP
